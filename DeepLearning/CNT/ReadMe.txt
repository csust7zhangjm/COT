This code is a MATLAB implementation of the tracking algorithm described in TIP paper 
        "Robust Visual Tracking via Convolutional Networks without Training" 
               by Kaihua Zhang, Qingshan Liu, Yi Wu and Ming-Hsuan Yang.


***********************************************************************
The code runs on Windows XP with MATLAB R2013a.

This code runs under the CVPR 2013 tracking benchmark library (http://cvlab.hanyang.ac.kr/tracker_benchmark/).
